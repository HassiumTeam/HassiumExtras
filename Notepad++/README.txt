Hassium UDL plugin for Notepad++
-- by zdimension

HOW TO INSTALL:
1. Run Notepad++
2. Click on the "Language" menu
3. Click on "Define Your Language"
4. Click "Import" and select hassium_npp.xml
5. Restart Notepad++
6. Make sure "Hassium" is present at the bottom of the "Language" menu. If it's not here, contact us.
7. Enjoy! As of now, any .has file will have nice Hassium syntax highlighting.
